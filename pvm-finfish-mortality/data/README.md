# Data folder

Place the project input CSV files here before running the analysis. The draft code expects these files:

- `Mortality_data_for_full_production_cycles_2018-2024_v2.csv`
- `FHI 066 Mortality events 16092025_grouped_causes.csv`
- `SEPA_mortality_all_data.csv`
- `StockingFallowMorts-SEPA_checked.csv`
- `data_for_app_development.csv` for the Shiny app

Raw data are not included in this repository template because they may be large, unpublished, or sensitive.
